发表于 2017-04-29

0x0

首先需要确保你的 Xshell 的连接编码是 UTF-8。
虽然可以修改 Linux 端的字符集去迎合 Windows 端做到中文的正常显示，但在当前 I18N 的大环境下，还是尽量用 UTF-8 吧。

打开 Xshell 的会话属性窗口，在左边选择到 终端，编码一项内，如果不是 Unicode 的话请改成 Unicode
